# Rspack Vue SSR Example

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https%3A%2F%2Fgithub.com%2Fupupming%2Frspack-ssr-examples%2Ftree%2Fmain%2Fpackages%2Fvue)

## Develop

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Preview

```bash
pnpm preview
```

## Deploy

```bash
vercel --prod
```
